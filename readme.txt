Mongoose Express CRUD Mastery
Follow the steps below to run this server locally

Download & intall Node.js . 
git clone https://github.com/purnoy/assignment_ph_crud.git
cd into directory and run
- npm install

- Build Server Command - npm run build
- Start The Server Comman - npm run start:dev 


- Eslint error - npm run lint

- Fixing eslint - npm run lint:fix


Routes -
Create a new user
Endpoint - POST /api/users

Retrieve a list of all users
Endpoint - GET /api/users

Retrieve a specific user by ID
Endpoint - GET /api/users/:userId

Update user information
Endpoint - PUT /api/users/:userId

Delete a user
Endpoint - DELETE /api/users/:userId

Add New Product in Order
Endpoint - PUT /api/users/:userId/orders

Retrieve all orders for a specific user
Endpoint - GET /api/users/:userId/orders

Calculate Total Price of Orders for a Specific User
Endpoint - GET /api/users/:userId/orders/total-price
